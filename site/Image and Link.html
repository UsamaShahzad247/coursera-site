Steps for adding an image to your website
  1. Open any Image on GOOGLE CHROME
  2. Right Click on it
  3. Copy Image reference
  4. If this option is not available(follow the below step)
    1. Right Click on Image and open inspect element
    2. There yous should find "src"
    3. Copy the link from there
  5. Now write command <src="paste the link here" height="as you like" width="as you like" >
  6. HURRAY THE IMAGE IS ATTACHED TO YOUR HTML SITE
